# Eco Picker

## Prerequisites

- IDE
- Flutter SDK
  If you are a mac user, you need to download:
- Xcode
- CocoaPods

## Getting Started

### git clone

```shell
git clone https://github.com/Eco-Picker/eco-picker.git
cd eco-picker
```

### launch flutter

```shell
flutter run
```
